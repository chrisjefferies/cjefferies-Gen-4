about this project

The State News designed and built their own custom content management system, Gryphon, in 2011. Since then, they've been steadily growing a full service web design agency. SNWorks, a division of State News, Inc, was formed in 2012. Originally comprised of a group of core developers, I was hired as the first dedicated creative.

With every client, we improved as a team. By the time the Wildcat came up, I had designed an implemented a standard set of icons to communicate responsive break points and animations to the developer team. To my knowledge, they're still in use.

a look at the process

designing for a cms

Indicating changes in the size of elements with browser size for the development team was easy enough. The challenge came persistently from designing elements that would feel the same on all devices. The solution was in a set of standardized proportions and relationships, along with delicate attention to typography.
